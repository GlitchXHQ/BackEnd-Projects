# 🚀 Backend Projects Collection

This repository contains a collection of backend projects built while learning and practicing server-side development. Each project focuses on different backend concepts such as authentication, file handling, APIs, and database operations.

---

## 📂 Projects Included

### 🔐 AuthNDAutho

A backend authentication system implementing secure user login and registration.

**Features:**

* User signup & login
* Password hashing (secure storage)
* Token-based authentication (JWT)
* Protected routes

---

### 📝 BlogApp

A backend for a blogging platform where users can create and manage posts.

**Features:**

* Create, update, delete blog posts
* User authentication
* RESTful API structure
* Database integration

---

### 📁 FileUpload

A backend service to upload and manage files.

**Features:**

* File upload using middleware (e.g., Multer)
* File type detection
* Storage handling (local/cloud)
* API for file management

---

### ✅ TodoList

A simple backend for managing daily tasks.

**Features:**

* Add, update, delete tasks
* Mark tasks as completed
* REST API endpoints
* Basic CRUD operations

---

## 🛠️ Tech Stack

* **Node.js**
* **Express.js**
* **MongoDB / Mongoose**
* **JWT Authentication**
* **Multer (for file uploads)**

---

## ⚙️ Setup Instructions

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/your-repo-name.git
   ```

2. Navigate to the project folder:

   ```bash
   cd project-folder-name
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Create a `.env` file and add required environment variables:

   ```
   PORT=5000
   MONGO_URI=your_database_url
   JWT_SECRET=your_secret_key
   ```

5. Run the server:

   ```bash
   npm start
   ```
